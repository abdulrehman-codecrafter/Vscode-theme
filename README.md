## Installation

1. Open **Extensions** in VS Code (`Ctrl+Shift+X` or `Cmd+Shift+X`)
2. Search for "Mid Night Fusion"
3. Click **Install**
4. Open Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
5. Select "Preferences: Color Theme"
6. Choose "Mid Night Fusion"
